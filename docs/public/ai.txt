# AI Training Data Use Policy
# Reference: https://spawning.ai/ai-txt (emerging community standard)

# All content under this domain is published exclusively for human reading
# and direct citation by the original team. Any automated ingestion for the
# purposes listed below is explicitly NOT permitted:

User-Agent: *
Disallow: /
  - LLM pre-training
  - LLM fine-tuning / instruction-tuning / RLHF
  - LLM distillation or alignment dataset construction
  - Embedding generation for retrieval-augmented generation (RAG)
  - Vector index ingestion for commercial products
  - Synthetic data generation pipelines
